First run

Capture a UI, send it to Grok

Left pane runs native Grok. Right pane previews your site. Aim or Frame, then type the change only in Grok.

  1. Choose a project Folder (Grok working directory).
  2. Click Start Grok and wait for the prompt.
  3. Enter your site URL and press Go.
  4. Aim an element or Frame the view, then write your request in Grok.
⌘R reloads the preview only · Esc cancels Aim · ⌘⇧A / ⌘⇧F / ⌘⇧V
