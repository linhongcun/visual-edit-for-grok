用视觉方式编辑前端

这是一个本地演示页。在 Visual Edit for Grok 中打开本页，拾取任意元素，然后让 Grok 修改样式或文案。

拾取元素

在右侧预览里点选任意 DOM 节点，一次采集定位信息与视觉上下文，再交给左侧 Grok 精准改稿。

  • 自动生成 CSS 选择器、DOM 路径、tag / id / class、可见文案与边界框
  • 拾取时默认截图：优先裁切元素区域，并保留整页备份
  • 输出对齐 Cursor browser_element，可附带样式 before→after 与改稿意图
  • 结果写入剪贴板，并可自动粘贴进原生 Grok 终端

Grok Build

左侧嵌入原生 grok TUI（真实 shell），采集结果直接投递到输入区，由 Grok 在项目源码里改样式与文案。

  • 一键启动 Grok，终端 cwd 对齐你的前端项目目录
  • 意图面板可描述「按钮再大一点」等改动，并编辑目标样式
  • 不另做一套 agent 聊天，就是原生 Grok Build 工作流

前端友好

左右分屏：终端 + 本地预览，从点选到改稿一条闭环，适合 Vite / React / Vue 等日常前端开发。

  • BrowserView 打开本地预览 URL，中间分隔条可拖动
  • 截图落盘 ~/.grok/visual-edit-captures/，外置终端也能 ⌘V
  • 自动粘贴可开关；可随时再次「粘贴到终端」重投上次采集
